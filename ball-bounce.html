Simple Ball Bounce Program

Importing the Necessary Module

The following line imports the pgzrun module which is used to run Pygame Zero games:


    import pgzrun

Setting Up the Window

Width and Height of the game window are set by the following lines. These dimensions are measured in pixels:


    WIDTH = 600
    HEIGHT = 450

Configuring the Ball Properties

Here, we define a dictionary ball to manage and contain all properties related to the bouncing ball:


    ball = {
        'x': WIDTH // 2,
        'y': HEIGHT // 2,
        'radius': 30,
        'color': (255, 0, 0),
        'dx': 5,  # x velocity
        'dy': 5   # y velocity
    }

Updating Ball Position

The update() function refreshes the game state. In this section, it adjusts the ball's position based on its current velocity:


    def update():
        ball['x'] += ball['dx']
        ball['y'] += ball['dy']

Implementing Bounce Logic

Continuing within the update() function, logic is applied to make the ball bounce upon hitting the window's borders by inverting its velocity:


        if ball['x'] < ball['radius'] or ball['x'] > WIDTH - ball['radius']:
            ball['dx'] = -ball['dx']
        if ball['y'] < ball['radius'] or ball['y'] > HEIGHT - ball['radius']:
            ball['dy'] = -ball['dy']

The draw() Function

The draw() function refreshes the screen during every animation frame and here it draws the ball in its new position:


    def draw():
        screen.clear()
        screen.draw.filled_circle((ball['x'], ball['y']), ball['radius'], ball['color'])

Running the Game

The game is set into motion with the following line, which keeps it running in a continuous loop, invoking the update() and draw() functions repeatedly:


    pgzrun.go()
