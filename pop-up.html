Import pgzrun

In order to run our game, we need to import pgzrun. Add the below code. 
import pgzrun

Create the Actor

In pygameZero, an actor is any character, item, or any visible element in the game. They are drawn onto the screen at any postion the programmer specifies. Let's create an actor called "person" using the "person" image file that we loaded inot our project and then sets the initial position of the person actor at coordinates (100, 150). 

    person = Actor("person")
    person.pos = 100, 150

Draw the Screen and the Actor

Next, we need to use the draw() function to display the Actor on the screen, change the background color, and display a message

Let's first define the draw() function which is responsible for drawing the screen and the actor. A function is essentailly a set of code that other parts of a prorgam can execute with out having to re-write the code (like a shortcut!) Note that functions in pygame zero (and python) are defined by a name immediately followed by parentheses and a colon ():. All of the code tied to a function is written underneath the funation name and is indented relative to the function name. 

    def draw():
Next, we fill the background of the screen with the color blue--make sure the below line of code is indented relative to the function name. 

        screen.fill("blue")
Then, draw the text "Welcome to my game" at the center coordinates (150, 75) with font size 40 and color white. 

        screen.draw.text("Welcome to my game", center =(150,75), fontsize=40, color="white")
Finally, to complete the initial setup of our game, we draw the person actor on the screen. 
    
        person.draw()

Update the game

The update() function handles the game logic, such as actor movement. Type in the following code to define the update() function.

Define the update() function. 

    def update():
Specify that we are using the global variable "person". A global variable is a variable that is accessible from anywhere in the code. In Python, we can use the `global` keyword inside a function to indicate that you are referring to a global variable rather than a local variable with the same name (in this case, person) 

        global person
Next, we have to tell the game what to do if the left arrow key is pressed. What it will do is move the person actor 5 units to the left, by subtracting 5 from its current x positon. 

        if keyboard.left:
            person.x -= 5
Next, we have to tell the game what to do if the right arrow key is pressed. What it will do is move the person actor 5 units to the right, by adding 5 to its current x positon. 

        if keyboard.right:
            person.x += 5
Next, we have to tell the game what to do if the up arrow key is pressed. What it will do is move the person actor 5 units up, by subtracting 5 from its current y positon. 

        if keyboard.up:
            person.y -= 5
Next, we have to tell the game what to do if the down arrow key is pressed. What it will do is move the person actor 5 units down, by adding 5 to its current y positon. Notice how for up, we subtractm and down, we add 

        if keyboard.down:
            person.y += 5
To wrap up the update function, we have to tell the game what to do if the enter key is pressed. What it will do is swap the person actor to use the person2 image in place of the initial person image and reset the person actor's position to coordinates (100, 150) 

        if keyboard.RETURN:
            person = Actor("person2")
            person.pos = 100, 150

Call pgzrun.go()

In order to start the game, we have to use our pgzrun import by calling pgzrun.go() at the very end of our program. Add the following line to your program. MAKE SURE IT'S ALWAYS THE LAST LINE OF CODE IN YOUR PROGRAM. 

    pgzrun.go()
